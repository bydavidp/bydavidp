## Sobre mí

Soy estudiante de **Ingeniería en Sistemas** en la **Universidad del Valle** (Cali, Colombia) y desarrollador de software en formación.  
Me interesa construir aplicaciones web bien diseñadas, con foco en arquitectura, seguridad y calidad de código.

- Experiencia académica y personal en **Flask**, **C++ orientado a objetos**, bases de datos y sistemas web.
- Interés en **seguridad web**, herramientas de análisis de vulnerabilidades y buenas prácticas **OWASP**.
- Exploración de **IA aplicada**: asistentes tipo Jarvis, automatización de tareas y modelos locales vía Ollama.
- Proyectos personales que combinan gestión empresarial, herramientas de seguridad y plataformas de contenido.

## Stack Tecnológico

### Lenguajes

- C++ (POO, herencia, polimorfismo, abstracción)
- Python (scripts, backend con Flask, automatización)
- JavaScript (frontend básico y lógica en navegador)
- SQL (consultas, diseño de tablas, ORM con SQLAlchemy)
- HTML/CSS (estructuras y estilos responsivos)
- PHP (bases para backend tradicional)

### Backend & Frameworks

- Flask (rutas, templates, auth, integración con ORM)
- SQLAlchemy (modelado de datos, relaciones, consultas)
- Django (conocimientos base del framework)
- Node.js (scripts y pequeños servicios)

### Frontend y UI

- Bootstrap (componentes, grid, estilos responsivos)
- Jinja2 (renderizado de vistas en Flask)

### Bases de datos

- SQLite (prototipos y proyectos académicos)
- MySQL (entornos más estructurados de base de datos)

### Herramientas y entorno

- Git y GitHub (control de versiones, flujo de trabajo con ramas)
- Docker (contenedores básicos para servicios)
- Linux / WSL (entorno de desarrollo y comandos)
- VS Code / CLion (IDE para C++ y proyectos multiplataforma)
- CMake (configuración y construcción de proyectos C++)

### Seguridad e IA

- Kali Linux y OWASP (introducción a herramientas y metodologías)
- Ollama (modelos locales de IA)
- Asistentes de IA (Claude, otros) como apoyo al flujo de desarrollo
